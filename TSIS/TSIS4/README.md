# TSIS4 Racer

A complete Pygame project with:
- main menu
- username entry
- settings screen
- leaderboard screen
- game over screen
- traffic cars, lane hazards, dynamic road events
- Nitro, Shield, and Repair power-ups
- saved settings and top scores in JSON

## Run
```bash
pip install -r requirements.txt
python main.py
```

## Files
- `main.py` — app entry point and screen flow
- `racer.py` — race gameplay and spawning logic
- `ui.py` — buttons and text input
- `persistence.py` — JSON save/load helpers
- `settings.json` — saved preferences
- `leaderboard.json` — saved scores
